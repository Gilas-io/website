تکمیل خودکار کد با Gilas API و Continue.dev

درین راهنما نحوه تنظیم افزونه‌ی محبوب Continue.dev برای تکمیل خودکار کد code auto-completion که گاهی از آن به عنوان Tab completion هم یاد می‌شود را با استفاده از Gilas API بررسی میکنیم.

اتصال ادیتور Cursor به Gilas API

اگر از ادیتور محبوب Cursor برای کدنویسی استفاده می‌کنید٬ خبر خوب این است که می‌توانید آن را به APIهای گیلاس متصل کرده تا به مدل‌های OpenAI از جمله GPT-4o و o1 در محیط ادیتور دسترسی داشته باشید.

اتصال فریم ورک ها و SDK ها به API گیلاس

در این پست کوتاه، نحوه پیکربندی فریم‌ورک‌ها و SDKهای مختلف از جمله LangChain, Microsoft AutoGen و OpenAI SDKs برای دسترسی به API گیلاس را بررسی می‌کنیم.

ساخت چت‌ بات با استفاده از LangChain

در این راهنما با مثالی از طراحی و پیاده‌سازی یک چت‌بات با استفاده از LangChain و Gilas API آشنا می‌شوید که از قابلیت هایی مثل Agent و RAG که از طریق LangChain قابل دسترس هستند استفاده می‌کند.
